# ABOUT DICTATOR

DICTATOR is a complete Cyber Security Resource Repository created to help beginners, students, researchers, security enthusiasts and professionals learn Cyber Security through a structured path.

This repository contains carefully organized notes, roadmaps, guides, references, tools, labs and writeups covering almost every major domain of Cyber Security.
